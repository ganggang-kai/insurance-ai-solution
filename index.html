一、项目概述

本项目旨在构建一套「AI外呼筛意向 → 公众号自助投保」的智能化客户转化体系。 利用AI语音机器人完成批量外呼，精准识别意向客户后，通过专属码机制将客户无缝引导至微信公众号， 在公众号内完成信息填写、健康告知、在线支付，最终实现保单生效。 全程数据埋点可追踪，AI根据客户行为实时调整引导策略，最大化提升转化率。

三、核心业务流程

全链路共7个阶段、18个关键节点，AI根据客户行为实时调整策略：

  1. 📞 接通建信

    AI外呼确认本人接听 → LLM语义筛选意向 → LLM判断信任度。冷启动场景下自动判别客户质疑类型并分流，诈骗类直接转人工。

  2. 📲 微信搜索与关注

    语音引导客户搜索公众号 → API定时查询关注状态（5秒/次，最长等待60秒）→ 未关注则降级为短信直链。

  3. 🔑 专属码交付

    口播四位专属码 + 同步短信发送。客户在公众号输入后自动反显预留手机号，零手动输入。

  4. 🎯 入口行为引导（埋点轮询核心）

    AI实时查询公众号埋点（5秒/次，最长120秒），根据客户行为分级介入——15秒无操作提示、30秒换方式引导、120秒短信兜底。

  5. 📝 信息填写与健告

    轮询确认姓名身份证填写完成 → 引导增员（家庭版）→ 调用核保API判断健告结果。拒保自动转人工推荐替代产品。

  6. ⚕️ 支付与成交

    查询支付状态 → 余额不足引导换卡 → 犹豫客户使用「犹豫期内可全额退款」减压 → 超时30分钟自动短信追回。

  7. 🏁 闭环与延伸

    投保成功 → 告知保障生效时间 → 引导查看保单 → 推荐家庭版加保入口。

5 短信模板配置 设计并配置7条场景化短信模板：未接通触达、无意向留存、降级直达链接、入口超时兜底、信息填写提醒、支付追回、投保成功通知 短信模板清单 + 发送策略表

7 联调测试 端到端全链路测试：外呼→公众号关注→投保→支付→保单生效。覆盖正向流程、各负向分支、短信降级、转人工升级 测试用例 + 测试报告
8 上线后运营支持 提供首月运营数据分析、转化漏斗监控、话术A/B测试建议、画布节点迭代优化 月度运营分析报告

八、方案核心优势

🛡️

合规可控

话术预设在画布普通节点中，不做实时生成，每条话术可审计可追溯。LLM仅用于语义分类分流，不直接面向客户输出内容。

🔄

闭环不中断

客户挂机后短信链路可自闭环完成投保，不依赖AI持续在线。每个流失节点都有回收机制，最大化转化漏斗。

⚡

快速迭代

画布节点可独立修改不互相影响，话术可逐节点A/B测试。非技术人员也可调整流程参数（超时时间、重复次数等）。

📈

效果可量化

全链路埋点+转化漏斗报表，每个节点流失率可量化。可精准定位流失瓶颈并针对性优化话术或流程参数。
