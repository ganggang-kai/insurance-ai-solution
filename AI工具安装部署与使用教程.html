三 · 安装前的准备工作

不管你要用哪个工具，先装好这 三样基础依赖 准没错。后面每个工具的安装章节也会引用这里。

① 安装 Node.js（v18 或更高版本）

为什么装？ VS Code 的 AI 插件、Claude Code CLI、Codex CLI 都需要 Node.js 才能运行。

  1. 打开 nodejs.org
  2. 下载左侧 LTS（长期支持版），不要下载右侧的 Current 版
  3. 双击安装包，一路点"下一步"（Windows 用户注意勾选"Add to PATH"）
  4. 安装完成后，打开终端验证：
node -v
npm -v

如果显示版本号（如 v18.20.0），说明安装成功。

② 安装 Git + Git Bash（Windows 用户必看）

为什么装？ Git 是代码操作的基础工具；Git Bash 是 Windows 下一个好用的终端环境。

  1. 打开 git-scm.com
  2. 下载 Windows 版本，双击安装（一路默认设置即可）
  3. 安装完成后，在桌面右键 → 点击 "Git Bash Here"
git --version
💡 macOS 用户：macOS 自带的终端已经很好用。Git 可以通过 xcode-select --install 安装。

③ （可选）安装 Python

为什么装？ AI 生成的运营脚本（数据分析、报表生成）很多用 Python 写。

  1. 打开 python.org
  2. 下载最新版，安装时务必勾选 "Add Python to PATH"
python --version

④ 准备 API Key

如果你要使用 Claude Code、Codex CLI 或 Hermes Agent，需要准备一个 API Key。

模型提供商获取方式费用国内可用是否需要魔法？
DeepSeek（推荐） platform.deepseek.com → 注册 → 创建 API Key 极低，约 ¥1/百万 token ✅ 直接可用 ❌ 不需要
智谱 AI（GLM） open.bigmodel.cn → 注册 → 创建 API Key 低，有免费额度 ✅ 直接可用 ❌ 不需要
通义千问（Qwen） dashscope.aliyun.com → 注册 → 创建 API Key 低，有免费额度 ✅ 直接可用 ❌ 不需要
OpenAI platform.openai.com → 注册 → 创建 API Key 中等，约 $5-10/月 ✅ 可直接访问 ⚠️ 注册时可能需要
Anthropic console.anthropic.com → 注册 → 创建 API Key 较高，约 $10-20/月 ⚠️ 有封号风险 ✅ 需要魔法
💡 初学者建议：先注册 DeepSeek 或 智谱 AI，都是国内直连、无需魔法、充值几块钱就够用很久。即使后面要用其他工具，也可以作为备用模型。

四 · VS Code 安装与 AI 插件配置

VS Code 是微软出品的一款 免费、开源 的代码编辑器，是目前全球最流行的开发工具。装上 AI 插件后，它就变成一个 AI 编码助手。

4.1 下载安装 VS Code

  1. 打开 code.visualstudio.com
  2. 点击下载按钮，双击安装包，一路点"下一步"
  3. 安装完成后启动 VS Code

4.2 切换为中文界面

  1. 按快捷键 Ctrl+Shift+X（Mac：Cmd+Shift+X）打开扩展市场
  2. 搜索 Chinese → 找到 Chinese (Simplified) Language Pack → Install
  3. 右下角提示重启，点击 Restart

4.3 安装 AI 插件（国产推荐方案）

插件名称开发方价格能力需要魔法？
通义灵码阿里云免费代码补全 + AI 对话 + 代码生成，中文支持好❌ 不需要
CodeGeeX智谱 AI免费代码生成 + 翻译 + 注释❌ 不需要
GitHub Copilot微软/GitHub$10/月目前最强的 AI 代码补全⚠️ 可能需要

安装方法（以通义灵码为例）：Ctrl+Shift+X → 搜索 "Tongyi Lingma" → Install → 侧边栏出现图标即可对话

4.4 VS Code 中使用 Claude Code 插件

Claude Code 有独立的 VS Code 插件，在扩展市场搜索 "Claude Code by Anthropic" 安装即可。

💡 不需要额外工具
Claude Code 的 VS Code 插件 内置了 CLI，安装后直接在 VS Code 中使用，不需要单独安装 CC Stitch 或任何其他辅助工具。你只需要一个 Claude 订阅账号（Pro / Max / Team / Enterprise）或 Console API Key 即可登录使用。

前提条件：VS Code 版本需要 1.98.0 以上。

4.5 验证：写第一段 AI 辅助的脚本

  1. 点击通义灵码图标，在对话框中输入：
    "写一个 Python 脚本，读取一个 CSV 文件，统计每列的数据分布，输出为 Markdown 表格"
  2. AI 会生成代码，点击 "插入到编辑器" 按钮
  3. 保存为 analyse.py
  4. 按 Ctrl+` 打开 VS Code 内置终端，运行：
python analyse.py
💡 全程不需要手动复制粘贴代码——AI 生成的代码可以直接插入编辑器，在 VS Code 内置终端运行。

五 · Claude Code 安装与使用

Claude Code 是 Anthropic（Claude 的母公司）推出的 AI 编码代理。你告诉它需求，它直接读写你的文件、运行命令、管理代码库。

⚠️ 国内用户重要提示
Anthropic 对来自中国大陆 IP 和支付方式的账号有 封号风险。使用前请评估：
• 需要 魔法 才能正常使用
• 即便有魔法，也可能因为 IP 变动、支付方式等被封号
• 建议优先考虑其他工具，或通过第三方 API 网关降低风险

5.1 安装方式一：桌面客户端（推荐新手）

  1. 打开 claude.ai/download（需魔法）
  2. 下载对应系统的版本
  3. 双击安装，登录你的 Claude 账号
  4. 点击 Code 标签进入编码模式

5.2 安装方式二：终端 CLI

npm install -g @anthropic-ai/claude-code

5.3 降低封号风险的配置方案

Claude Code 支持通过 ANTHROPIC_BASE_URL 环境变量将请求路由到第三方 API 网关：

# Windows（PowerShell）
$env:ANTHROPIC_BASE_URL="https://your-gateway.com"

# macOS / Linux
export ANTHROPIC_BASE_URL=https://your-gateway.com

5.4 验证：第一条运营指令

启动 Claude Code 后，输入：

"帮我写一个 Python 脚本，从 CSV 文件中统计用户反馈的分类占比，并生成柱状图"

六 · Codex CLI 安装与使用

Codex CLI 是 OpenAI 开源的编码代理（GitHub 94k+ stars）。

6.1 安装方式一：桌面 App（推荐新手）

方法 A：npm 安装后启动

npm install -g @openai/codex
codex app

方法 B：GitHub Release 下载二进制（推荐公司设备）

  1. 打开 github.com/openai/codex/releases/latest
  2. 下载对应系统的压缩包
  3. 解压后直接运行 codex
⚠️ 公司设备避坑
在公司电脑上通过 npm 安装后，首次运行 codex app 会自动打开微软商店下载组件（走系统更新路径），而公司 IT 策略通常会锁定注册表修改权限，导致安装失败。

解决方案：改用 GitHub Release 二进制文件 或安装 VS Code / Cursor 插件版。

6.2 安装方式二：IDE 插件

在 VS Code 扩展市场搜索 Codex by OpenAI，或访问 developers.openai.com/codex/ide

6.3 登录认证

  • ChatGPT 账号（推荐）：选择 "Sign in with ChatGPT"
  • API Key：选择 "Sign in with API Key"

6.4 验证：第一条运营指令

"分析 CSV 中的用户反馈数据，分类统计并生成一份运营周报"

七 · Hermes Agent 安装与使用

Hermes Agent 是 Nous Research 推出的 自主 AI 智能体。和前面几个工具不同，它最大的特点是：它会自我学习、能定时自动执行任务、还能把结果推送到你的微信/飞书/Telegram。

7.1 桌面版安装

  1. 打开 hermes-agent.nousresearch.com
  2. 点击 Download，下载对应系统的安装包
  3. 双击安装即可
⚠️ 安装时的网络要求
Hermes Agent 桌面版 下载简单（下载安装包本身不大），但 安装过程中 需要联网下载依赖组件，对网络稳定性要求较高。

国内网络不稳定？ 建议安装过程中 开启魔法，确保安装顺利完成。安装成功后的日常使用不需要魔法（如果你用 DeepSeek 等国内模型）。

7.2 CLI 安装（高级选项）

# Windows（PowerShell）
iex (irm https://hermes-agent.nousresearch.com/install.ps1)

# macOS / Linux
curl -fsSL https://hermes-agent.nousresearch.com/install.sh | bash

7.3 配置模型——接入 DeepSeek

  1. 启动 Hermes，进入设置页面
  2. 找到模型配置，设置：
    Provider: deepseek
    Model: deepseek-chat
    API Key: 填入你在 DeepSeek 平台获取的 Key
  3. 保存后即可开始对话

7.4 核心概念扫盲（用大白话解释）

术语大白话解释
Skills（技能）像给 Hermes 一本"操作手册"。你教它怎么做一件事，它会记住并下次自动应用。比如教它"竞品周报的格式"，以后每次出周报都用这个格式。
Memory（记忆）Hermes 的"笔记本"。它会记住你的偏好、项目信息、历史对话，不需要你反复说。
Cron（定时任务）像设闹钟。告诉 Hermes "每周一早上9点执行 XX 任务"，到时候它会自动干活。
MoA（多模型协作）像"开团队会议"。同时让多个 AI 模型讨论同一个问题，综合出最好的答案。
SOUL.md（灵魂文件）Hermes 的"人格定义文件"。你可以通过 SOUL.md 定义 Hermes 的身份、性格、沟通风格和行为准则。就像给 AI 写一份"角色说明书"——它按这份说明书来和你互动。不同的 SOUL.md 就像不同的 AI 角色。

7.5 创建你的专属 AI 角色：SOUL.md 实战

这是 Hermes Agent 最有趣的功能之一——你可以通过对话引导 Hermes 自己生成一份 SOUL.md 文件，定义出你想要的 AI 角色。

操作步骤

打开 Hermes Agent 对话框，直接输入以下提示词（可以根据你的需求修改方括号中的内容）：

💬 你可以直接复制这段提示词：

我想创建一个新的 AI 角色（Profile），请引导我完成。

这个角色的名字是：[贾维斯]
它的主要职责是：[作为我的工作助手，在我需要完成某一个项目的工作规划和落地方案时，可以跟我一起脑暴，可以很好的引发和引导我思考，并配合我产出交付。]

请先帮我起草一份 SOUL.md 文件的内容。
这份文件需要定义这个角色的：
• 身份与性格：它是谁？有什么样的性格特点？
• 沟通风格：它是直接还是委婉？是技术严谨还是富有创意？
• 行为准则：它应该避免什么？有什么核心原则？

起草好后，请向我提问，并根据我的回答来调整和完善这个角色定义。

效果说明

  1. Hermes 会根据你的描述，先生成一份 SOUL.md 草稿
  2. 然后它会反过来向你提问（比如"你希望它更活泼还是更严肃？"、"遇到分歧时它应该坚持己见还是听从你？"）
  3. 你回答后，它会自动修改并完善 SOUL.md
  4. 最终你确认满意后，它会帮你保存这份 SOUL.md 到对应的 Profile 目录
💡 你可以创建多个角色
比如：一个严谨的"代码审查官"、一个活泼的"脑暴搭子"、一个沉稳的"项目经理"。通过切换 Profile 来使用不同角色。

7.6 实战：让 Hermes 每周自动出竞品周报

  1. 向 Hermes 描述你的需求：
    "请你每周一早上9点，帮我搜索竞品动态，分析趋势，按照我教你的格式生成周报，发到我微信"
  2. Hermes 会创建 Cron 定时任务
  3. 确认后，从此每周自动执行

八 · WorkBuddy 安装与使用

WorkBuddy（腾讯龙虾）是腾讯推出的 全能 AI 工作台——不需要魔法、不需要装任何依赖、不需要写代码，下载安装后直接就能用。

8.1 下载安装

  1. 打开 workbuddy.tencent.com

十 · 魔法上网教程

部分工具（Claude Code、Hermes Agent 安装过程）需要稳定的国际网络访问。以下是配置"魔法"的简明教程。

💡 什么是魔法？
"魔法"是一种网络加速方案，让你能够访问部分在国内无法直连的网站和服务（如 Google、GitHub、Anthropic 等）。本教程以 AgentNEO 服务为例。

10.1 注册 AgentNEO 服务

  1. 打开 neoproxy.me
  2. 注册账号并购买套餐（通常有轻量版可选）
  3. 进入套餐详情页面，记下你的「订阅链接」

10.2 Windows 平台配置（Clash Verge）

第一步：下载安装客户端

  1. 前往 Clash Verge Rev 的 GitHub Releases 页面
  2. 下载最新版 Clash.Verge_x.x.x_x64-setup.exe
  3. 双击安装，安装后打开 Clash Verge
💡 如果 GitHub 打不开，可以在 AgentNEO 官方教程中找到备用下载链接：neoproxy.me/doc

第二步：导入订阅

  1. 登录 AgentNEO，进入套餐详情 → 点击「配置下载（订阅）」
  2. 选择「增强规则订阅」→ 找到「Clash 配置链接」→ 点击「复制地址」
  3. 打开 Clash Verge → 点击左侧「订阅」
  4. 将复制的地址粘贴到输入框 → 点击「导入」
⚠️ 注意顺序！
在订阅配置成功之前，不要先打开「代理」开关！否则会导致订阅获取失败。

第三步：激活并使用

  1. 在订阅列表中，对新导入的配置 右键 → 选择「使用」
  2. 回到主页面，打开「系统代理」开关
  3. 此时你已经可以访问国际网络了 ✅

运行模式说明

模式说明推荐场景
规则模式（推荐）自动分流：国内流量直连，国外流量走代理日常使用，不影响国内网速
全局模式所有流量都走代理遇到特殊情况时临时切换
直连模式关闭代理，所有流量直连不需要魔法时使用
